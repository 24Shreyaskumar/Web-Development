Table, thead, tbody, tfoot, caption, tr, th, td :

->The table tag is used to create a table of size m x n, where m is the number of rows and n is the number of columns
->The thead indicates that a group of rows are header rows
->The tbody indicates that a group of rows are body rows
->The tfoot indicates that a group of rows are footer rows
->The caption tag is used to give name to the table
->The tr tag is used to create rows in the table
->The th tag is used to give headers to the columns in the table
->The td tag is used to add the content in the respective rows / columns

Cellpadding & Cellspacing, Rowspan & Colspan

->Cellpadding & Cellspacing are the table tag elements
->Cellpadding is used to give spaces b/w cell edges and cell content of a cell in table
->Cellspcaing is used to give space between each cell in table
->Rowspan and Colspan are td tag attributes
->Rowspan accepts a numeric values, e.g rowspan = 2, which means number of rows that particular table cell will span
->Similarly the colspan accepts a numeric value which specifies the number of columns that table cell will span
